# The Gulabi Store 🌸
### A Digital Marketplace for Jaipur's Artisans

A modern E-commerce platform built for my Major Project at Manipal University Jaipur. The Gulabi Store focuses on connecting local artisans with a global audience, specializing in handcrafted ceramics, textiles, and jewelry.

## Tech Stack
- **Framework:** Next.js 15
- **Database:** PostgreSQL (Prisma ORM)
- **Styling:** Tailwind CSS & ShadCN UI
- **Auth:** NextAuth.js

## Core Features
- **Artisan Focus:** Dedicated profiles for local makers.
- **Customization:** Ability to request personalized engravings/designs on products.
- **Admin Dashboard:** Full inventory and order management.
